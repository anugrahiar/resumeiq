## What is this?

I built ResumeIQ because I was tired of submitting resumes and hearing nothing back. The idea is simple — paste any job description, upload your resume, and get an ATS score with real feedback on what's missing and what's working.

No backend. No API keys. Everything runs in the browser using Puter.js.

---

## Features

- **ATS Scoring** — See how well your resume matches a job description
- **AI Feedback** — Get specific suggestions, not just a number
- **Resume Storage** — Keep all your resumes in one place
